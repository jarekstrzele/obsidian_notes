# Searching
`/something` search 'something'
`n` next found item (next 'something')
`N` back

**backwards**
`/?something`

select a token (normal mode) (exit) -> `#` or `*` 

## mark
`m` makr (in normal mode)
`ma` mark as `a`
to jump to this mark
` 'a `
