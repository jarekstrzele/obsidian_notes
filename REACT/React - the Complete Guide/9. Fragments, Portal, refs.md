
>[!important] JSX Limitations
>- you can return only one "root" JSX element (`React.createElement('h1',{},"main head")`)
>	- solutions:
>		- wrap in one tag e.g. `<div>`, 
>		- make your own wrapper

### your own wrapper
```jsx
const Wrapper = props => {
	return props.children
} ;

export default Wrapper;
```
