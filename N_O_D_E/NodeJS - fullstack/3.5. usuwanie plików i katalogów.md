### `rmdir` usuwa tylko puste katalogi
```js
fs.rmdir(path.join(__dirname, "todelete"), function(err){
    if(err){
        console.log(`błąd${err.message}`)
    } else {
        console.log("Usunięto folder") ;
    }
})
```
