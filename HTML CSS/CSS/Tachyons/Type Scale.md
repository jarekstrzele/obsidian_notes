# type scale

   Media Query Extensions:
     `-ns` = not-small
     `-m`  = medium
     `-l`  = large


You can use several scale type in one html tag
`<h1 class="f3 f-headline-m"`
`<h1 class="f3 f6-l"` normal `f3` but for larger screen smaller `f6`
`<p class="f2 f2-m f1-l "`
