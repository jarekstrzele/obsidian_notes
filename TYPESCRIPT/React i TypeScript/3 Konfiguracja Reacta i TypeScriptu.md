# tworzenie projektu
## użycie `vite`
- `npm create vite firstproject` - zaznacz `React`, `TypeScript`
- jeżeli są `waring`:
	- sprawdź przestarzałe pakiety `npm outdated`
	- zaktualizuj `npm update`
	- czasami trzeba usunąć `node_module` i `package-lock.json`
