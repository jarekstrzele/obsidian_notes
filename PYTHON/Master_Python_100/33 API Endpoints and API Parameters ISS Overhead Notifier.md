>[!info] API
>Application Programming Interface is a SET of:
>- commands
>- functions
>- protocols
>- objects
>
that the programmer can use to
> - create software or 
> - interact with an external system

> API is an **interface** or a sort of **barrier** between your *programm* and *external system*.
