# What is MongoDB?
- it is a database HUMONGOUS
- example:
	- database: *Shop*
	- collections: *Users*, *Orders*
	- documents: *{name: 'Max', age:29}*, *{name:'Max'}*
	- `BSON` format (*JSON* converted to a binary version)
```BSON
{
	"name":"Max",
	"age": 29,
	"address": 
	{
		"city":"Muchin"
	},
	"hobbies":[
		{"name":"Cokking"},
		{"name":"Sports"}
	]
}
```



# BSON data structure
- no schema
- no/few relations
