# Funkcje tekstowe

## UPPER
`SELECT UPPER(CategoryName) FROM Category;`

## LOWER
`SELECT LOWER(CategoryName) FROM Category;`

## LENGTH
`SELECT CategoryName, LENGTH(CategoryName) AS LenCategoryName FROM Category;`

## TYPEOF
`SELECT TYPEOF(3);`
`SELECT TYPEOF(NULL);`
`SELECT TYPEOF(categoryName) FROM Category;`
