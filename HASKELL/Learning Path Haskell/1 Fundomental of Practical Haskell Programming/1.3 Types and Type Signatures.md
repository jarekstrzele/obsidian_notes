
# Type synonums

```haskell
ghci> type Port = Int
ghci> :info Port
type Port :: *
type Port = Int
        -- Defined at <interactive>:1:1
```
`Port` is not a new independent type.  It is a alias to the `Int`
```haskell
ghci> type HostInfo = (String, Port)
ghci> :i HostInfo
type HostInfo :: *
type HostInfo = (String, Port)
```
