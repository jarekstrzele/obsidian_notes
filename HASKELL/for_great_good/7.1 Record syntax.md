[[7 Types and TypeClasses]]

OK, we've been tasked with creating a data type that describes a person. The info that we want to store about that person is: 
- first name, 
- last name, 
- age, 
- height, 
- phone number, and 
- favorite ice-cream flavor. 
I don't know about you, but that's all I ever want to know about a person. Let's give it a go!

```haskell
data Person = Person String String String Int Float String String deriving (Show)
```
==this is unreadable!!==

```haskell
data Person = Person { 
	 firstName :: String
	,lastName :: String
	, age :: Int
	, Height :: Float
	, phonNumber :: String
	, flavor :: String
	} deriving (Show) 
```

```haskell
-- Deklaracja modułu
module Main where

-- Importowanie modułów


-- Deklaracja funkcji
data Person = Person { 
  firstName :: String
 ,lastName :: String
 , age :: Int
 , height :: Float
 , phonNumber :: String
 , flavor :: String
 } deriving (Show) 
-- Funkcja main
main :: IO ()
main = do
let p = Person "tom" "jerry" 10 166.50 "234234234" "Fruity"
print p
```
