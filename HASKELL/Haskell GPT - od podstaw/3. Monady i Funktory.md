> Monady i funktory to zaawansowane struktury w Haskell, które pomagają w obsłudze efektów ubocznych (jak IO) w czysty sposób:

```haskell
-- Przykład użycia monady IO
main :: IO ()
main = do
    putStrLn "Jak masz na imię?"
    name <- getLine
    putStrLn ("Cześć, " ++ name ++ "!")

```

```haskell
-- Functor przykład
fmap (+3) (Just 5)  -- Wynik: Just 8

```
