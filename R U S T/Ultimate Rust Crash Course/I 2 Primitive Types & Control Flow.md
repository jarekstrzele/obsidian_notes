# Scalar types
- integer
- float
- boolean
- character


#### integer
unsigned: u8, u16, u32, u64, u128
sifned: i8, i16, i32, i64, i128

decimal: `1_000_000`
hex: `0xdead_beef`
octal: `0o7754_3211`
binary: `0b1111_0011`
Byte (u8 only): `b'A'`

#### float
```rust
let x: u16 = 5;
let y: f32 = 3.14;

//also:
let x = 5_u16;
let y = 3.14_f32 ;
```

`false`
